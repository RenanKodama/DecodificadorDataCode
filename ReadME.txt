Descrição:
	Os arquivos para testes devem estar contidos na pasta "Testes", pois é neste local
	que o programa irá procurar tais arquivos. Este trabalho foi desenvolvido na linguagem
	Python. Cada classe descrita no programa foi separada para decodificar pacotes especificos
	tais como, a classe "ethernetPack" que tem por finalidade decodificar o cabeçalho no formato
	Ethernet, a classe "ipPack" que tem por finalidade decodificar o cabeçalho IP e as classes 
	"udpPack" e "tcpPack" que decodificam os cabeçalhos no formato UDP e TCP respectivamente.

Funcionamento:
	Para executar o programa, primeiramente abra-o em um terminal e navegue até o local do arquivo
	"Decodificador.py", após feito isso, entre com o camando "$ python Decodificador.py", após inserir 
	o comando, será solicitado que entre com o nome do arquivo teste que será submetido ao programa. 
	Os resultados serão apresentados via terminal.
